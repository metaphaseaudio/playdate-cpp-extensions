Provides custom waveform generator functions for the synth.

Parameters
leftSampssample buffers in Q8.24 format
rightSampssample buffers in Q8.24 format if this is a stereo generator
nSampsthe number of frames to generate
ratethe amount to change a (Q32) phase accumulator each sample. Can be ignored if you're just using noteOn, but are required for any kind of transposition or modulation function to work smoothly.
dratethe amount to change rate each sample. Can be ignored if you're just using noteOn, but are required for any kind of transposition or modulation function to work smoothly.
Returns
the number of samples processed. Returning less than nSamps will inform the voice it's done playing, and this generator won't be called again until the next note-on.
