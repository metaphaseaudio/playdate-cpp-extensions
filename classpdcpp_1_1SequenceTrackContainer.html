◆ SequenceTrackContainer()

pdcpp::SequenceTrackContainer::SequenceTrackContainer ( ::SequenceTrack *  track)
explicit

Creates a new container for a PlaydateCAPI::SequenceTrack*, which comprises a PDSynthInstrument, a sequence of notes to play on that instrument, and any number of ControlSignal objects to control parameter changes.

These are generally owned by SoundSequence objects, but can be created stand-alone with the SequenceTrack class.
