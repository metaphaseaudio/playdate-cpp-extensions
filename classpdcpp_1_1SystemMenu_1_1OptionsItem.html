Creates a new item in the system menu which can be set to one of a number of values. Note that these options are displayed on the same line as the title, so space may be limited depending on the length of the title.

Parameters
titlethe title of the menu item
optionsa std::vector of std::strings to use as options
callbacka function which will be called with the string value and index of the selected option when the user interacts with the system menu item.
startingIndexwhich option to start with. defaults to 0
