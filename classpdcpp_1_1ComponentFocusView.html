◆ ComponentFocusView()

pdcpp::ComponentFocusView::ComponentFocusView ( )

Makes a ComponentFocusView. This component is like a viewport, but instead of one single content component, it can hold multiple Components, and provides an API which automatically brings one of the content components into view. This is used for classes like the GridView, where panning/scrolling is required to bring cells into view.
