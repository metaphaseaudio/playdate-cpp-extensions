◆ refreshComponentForCell()

virtual Component* pdcpp::GridView::refreshComponentForCell ( int  row,
int  column,
bool  hasFocus,
Component *  toUpdate 
)
protectedpure virtual

Override this to provide content for each of the GridView's cells. It will be called during refreshContent. Note that the GridView will not take ownership of the Component, so the user must manage its lifecycle.

Parameters
rowthe row of the cell to return
columnthe column of the cell to return
hasFocusindicate whether this cell has focus
toUpdatethe current Component in the cell, if any. Use this to either update the current component, or return a new one.
Returns
the component which will be displayed in the specified cell. if no updates are needed, you can return the exact same pointer as toUpdate.
