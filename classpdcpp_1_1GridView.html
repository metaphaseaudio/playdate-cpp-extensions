◆ refreshComponentForCell()

virtual Component * pdcpp::GridView::refreshComponentForCell ( int  row,
int  column,
bool  hasFocus,
Component *  toUpdate 
)
protectedpure virtual

Override this to provide content for each of the GridView's cells. It will be called during refreshContent. Note that the GridView will take ownership of the Component, so the user must not manage its lifecycle.

Parameters
rowthe row of the cell to return
columnthe column of the cell to return
hasFocusindicate whether this cell has focus
toUpdatethe current Component in the cell, if any. Use this to either update the current component, or return a new one. The GridView will own this component.
Returns
the component which will be displayed in the specified cell. if no updates are needed, you can return the exact same pointer as toUpdate.

Implemented in pdcpp::ListMenuComponent.

◆ setCellFocus()

void pdcpp::GridView::setCellFocus ( int  row,
int  column,
bool  shouldShowCell = true,
bool  limit = true 
)

Sets the cell at a given row/column as being in user focus. Optionally, if the cell is not currently in the Viewport, move that cell into the viewport.

Parameters
rowthe row of the cell for focus
columnthe column of the cell for focus
shouldShowCellwhether or not to move the Viewport to show the cell. On by default.
limitif limit is off, you can "deselect" by focusing on an element outside the range of the grid. The window will still scroll to the most relevant edge of the grid, but isFocused will be false for all elements when refreshComponentForCell is called. defaults to enabled.
