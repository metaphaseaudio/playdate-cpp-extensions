Detailed Description

A class to enable a "listener" pattern for button state modification. This patter allows the logic for button input management to be spread out across multiple classes, minimizing the need for a monolithic and repetitive button-management function/method in the main update routine.

Create one, add a bunch of listeners, and call checkStateAndNotify in your game's main update loop. All your listeners will be informed when the button state changes.
