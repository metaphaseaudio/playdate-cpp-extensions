Detailed Description

template<typename... Ts>
struct pdcpp::Overload< Ts >

Template class that allows std::visit to take a group of lambda functions

To use:

std::variant<int, float, std::string> myVariant;
std::visit(
Overload
{
[](int val) { ... },
[](std::string& val) { ... },
[](float val) { ... },
},
myVariant);
