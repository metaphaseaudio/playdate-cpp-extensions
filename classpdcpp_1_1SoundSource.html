virtual void pdcpp::SoundSource::finished ( )
pure virtual

Called when this sound source finishes playing. Note that, because this class is pure virtual, the callback must be set in the constructor of the inheriting classes. Do this by calling enableFinishedCallback in your constructors.

Implemented in pdcpp::Delay::Tap, pdcpp::SamplePlayer, pdcpp::Synthesizer, and pdcpp::SynthesizerVoiceContainer.
