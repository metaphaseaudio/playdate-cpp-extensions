pdcpp::FrameTimer::FrameTimer ( unsigned int  nFrames)
explicit

Similar to a Timer, but operates on a fixed count of ticks, as opposed to relying on the system time. This is ideal for triggering events on fixed frame counts, or reducing the effective refresh rate for sprites, etc.

Parameters
nFramesthe number of tick calls to wait before calling the timerCallback method.
