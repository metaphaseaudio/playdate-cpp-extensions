◆ TextKeyboard()

pdcpp::TextKeyboard::TextKeyboard ( const std::vector< char > &  toExclude = {})
explicit

Replicates the functionality of the Playdate Lua API's Keyboard. This is an input context and a Sprite, so it can take control of the user's input, and draw itself. Create one, set the callbacks, and use this as the input context to set up Keyboard interaction.

Parameters
toExcludea list of characters which should not be available in the keyboard
