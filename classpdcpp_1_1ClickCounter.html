Creates a ClickCounter. This class keeps track of all the new button presses for one or a combination of buttons within a window, and returns how many clicks were detected. It is very useful for detecting and handling double clicks, or more if your user interface requires it.

Parameters
toDetectthe button or button combination to detect
checkWindowthe window within which unique button presses will be counted
nClickMaxan optional early-return setting. ie. if you know you want to detect double clicks, set this to 2, and the detect method will return 2 after the second click, even if the window of detection hasn't closed yet. default is 0: "count them all"
